# Setup

## Install require, run in local

- ENV require

  - https://jekyllrb.com/docs/installation/
  - https://jekyllrb.com/docs/installation/ubuntu/

- Install bundle
  - `bundle install`
- Run in local
  - `bundle exec jekyll s`
